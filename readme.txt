== Description ==


**Custom Field For Easy Digital Downloads** you can be add in custom field in admin side as well frontend side job submit on show field. very simple way you need to add field just need to add field name which you want to add in [Easy Digital Downloads](https://easydigitaldownloads.com)

Ex. if you need to add salary, addition job field you can add in **Custom field Easy Digital Downloads**.

**Easy Digital Downloads field editor** can be add on field to action field name and placeholder in **Custom Field For Easy Digital Downloads**.

Normally by code you can add custom but by **Extra Field Easy Digital Downloads** using mostly for this plugin.

= Features =

&#9989; One Click installation
&#9989; Easy Digital Downloads Support in add field (Text, Select)
&#9989; Easy Digital Downloads Admin Fields in edit and update field
&#9989; Easy To use
&#9989; Auto Show Field on Single Product page

#### How to use Custom Field For Easy Digital Downloads
Go to admin in **Custom Field For Easy Digital Downloads > Add New Field** there just you need to add field name and text field will be show automatically in Easy Digital Downloads field in fronted and backend both side.
